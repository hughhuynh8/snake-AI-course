# Simple Snake Game

A classic Snake game built with Python's built-in `turtle` module. No external libraries required!

## Features
- Smooth movement using WASD or Arrow keys.
- Score tracking and High Score persistence (during the session).
- Increasing difficulty as you eat food.

## Terminal Version (Recommended)
If your Python environment doesn't support graphics (Tkinter/Pygame), run the terminal version:
```bash
python3 snake_cli.py
```
**Controls**: Use Arrow Keys to move. Press `Ctrl+C` or lose to exit.

## Graphical Version
If your environment supports it (requires Tkinter):
```bash
python3 snake.py
```

## Controls
- **Up**: `W` or `Up Arrow`
- **Down**: `S` or `Down Arrow`
- **Left**: `A` or `Left Arrow`
- **Right**: `D` or `Right Arrow`

## Goal
Eat the red food to grow and increase your score. Avoid hitting the walls or your own tail!
